## 結論

使用情境：

- 距離螢幕約 3 公尺
- 只買一台電視，客廳娛樂和電腦操作共用
- 電腦主機放電視旁，用 HDMI 直連
- 坐沙發用羅技無線鍵盤滑鼠操作
- 主要用途：Codex、VS Code、Markdown、瀏覽器、Windows、YouTube、Netflix
- 不能接受 OLED 烙印或畫面殘留風險

建議方向：

> 75 吋、4K、Mini LED / QD-Mini LED / Neo QLED、120Hz 以上、HDMI 2.1、PC Mode / Game Mode。

不要選 OLED。長時間顯示 Windows 工作列、VS Code、Codex、瀏覽器列，OLED / QD-OLED / WOLED 有永久烙印風險。Mini LED、QLED、LED LCD 本質是 LCD，比較適合當電腦螢幕。

## 賣場檢查清單

現場一定要問：

- 是否台灣公司貨
- 是否含基本安裝
- 壁掛安裝與線材是否另外收費
- 保固幾年，是否到府維修
- HDMI 2.1 有幾孔
- 能否接電腦跑 4K 120Hz / 144Hz
- 有沒有 PC Mode / Game Mode
- 是否支援 4:4:4 Chroma
- YouTube / Netflix 是否內建
- 手機能不能直接投 YouTube
- 是否能退換貨或現場驗機

現場實測：

- 看白底黑字，文字不能糊或有彩邊
- 移動滑鼠或切選單，延遲不能明顯
- 看賣場燈反光，不能像鏡子
- 從側邊看，顏色不要掉太多
- 開 YouTube / Netflix，系統不要卡
- 看暗場，光暈不要太誇張

## 壁掛高度

75 吋 16:9 電視大約寬 166 cm、高 93 cm。

你的使用方式是坐著看、坐沙發操作電腦，也偶爾站著看。壁掛高度應以坐著為主，站著為輔。

建議：

```text
螢幕中心高度：約 110 ~ 120 cm
螢幕底部離地：約 65 ~ 75 cm
螢幕上緣離地：約 158 ~ 168 cm
```

最簡單抓法：

> 75 吋壁掛時，底部離地約 70 cm。

不建議為了站著看，把中心拉到 140 cm 以上；坐著操作 Windows、Codex、VS Code 會變成長時間仰頭。

## Windows 建議設定

```text
解析度：3840 x 2160
Windows 縮放：200%
VS Code 字體：18 ~ 22 px
Terminal 字體：18 ~ 22 px
瀏覽器縮放：125% ~ 150%
Word 字體：16 ~ 20 pt
Excel 縮放：130% 左右
滑鼠速度：調快
工作列：可考慮放大或自動隱藏
```

電視端：

- 開啟 PC Mode 或 Game Mode
- 關閉動態補幀
- 關閉過度銳化
- 關閉降噪
- 亮度不要開太高
- 若有 HDMI 標籤設定，將接電腦的 HDMI 設為 PC
