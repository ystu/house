## 結論

使用情境：

- 距離螢幕約 3 公尺
- 只買一台電視，客廳娛樂和電腦操作共用
- 電腦主機放電視旁，用 HDMI 直連
- 坐沙發用羅技無線鍵盤滑鼠操作
- 主要用途：Codex、VS Code、Markdown、瀏覽器、Windows、YouTube、Netflix
- iPhone 也要能方便投放 YouTube、Netflix、照片或畫面
- 不能接受 OLED 烙印或畫面殘留風險
- 挑產品以 CP 值為主，不要太貴，也不要為了低價買品質與售後風險太高的品牌

建議方向：

> 75 吋、4K、Mini LED / QD-Mini LED、120Hz 以上、HDMI 2.1、PC Mode / Game Mode。

不要選 OLED。長時間顯示 Windows 工作列、VS Code、Codex、瀏覽器列，OLED / QD-OLED / WOLED 有永久烙印風險。Mini LED、QLED、LED LCD 本質是 LCD，比較適合當電腦螢幕。

Google TV 是加分，不是必要條件。只要硬體規格、PC 顯示效果、投放便利性和保固符合需求，VIDAA、Tizen、webOS 也可以納入候選。

## C7K 和 C6K 差異

兩台都符合主要需求：75 吋、Google TV、Mini LED、144Hz、HDMI 2.1、可接電腦。差價主要買到更好的背光、亮度、色彩、音響和系統餘裕。

| 項目 | TCL 75C7K | TCL 75C6K | 差異感受 |
|---|---|---|---|
| 價格 | 約 54,990 | 約 46,990 | C7K 貴約 8,000 |
| 背光分區 | 1248 區 | 512 區 | C7K 暗場、字幕光暈控制更好 |
| 峰值亮度 | HDR 3000 nits | 1000 nits | C7K 白天客廳與 HDR 更有餘裕 |
| 色域 | 97% DCI-P3 | 93% DCI-P3 | C7K 色彩更飽滿 |
| 儲存空間 | 3GB + 64GB | 3GB + 32GB | C7K 裝 App 與長期系統餘裕較好 |
| 音響 | B&O Audio System | 2.1 聲道 | C7K 音響定位較高 |

如果差價約 8,000，偏向選 `TCL 75C7K`。如果只是想壓預算、主要看文字和一般影音，`TCL 75C6K` 也可以。

## Google TV 的定位

Google TV 對這個情境的價值，是不用另外買電視盒，也能讓手機、影音 App 和電腦使用共存。但它不是必要條件；接電腦好不好用，主要還是看 HDMI 2.1、4K 120Hz、PC Mode、4:4:4 Chroma、Input lag 和面板。

| 功能 | 方便在哪 |
|---|---|
| YouTube 投放 | iPhone 或 Android 手機看到影片，直接投到電視 |
| Chromecast built-in | Chrome、Android 和支援 Cast 的 App 可直接投放 |
| App 生態 | YouTube、Netflix、Disney+、Prime Video、Spotify 通常都好裝 |
| Google 帳號同步 | YouTube 觀看紀錄、訂閱、推薦內容和手機/電腦接近 |
| 語音搜尋 | 用遙控器說片名或影片，比打字快 |
| iPhone 搭配 | 若電視支援 AirPlay 2，iPhone 相簿和螢幕鏡像也方便 |

TCL `75C7K` 和 `75C6K` 都是 Google TV，且規格標示支援 AirPlay 2，對 iPhone 也算友善。非 Google TV 機型只要 YouTube / Netflix 內建、投放功能可接受、iPhone AirPlay 或 App 投放沒問題，也能滿足需求。

## 賣場檢查清單

現場一定要問：

- 是否台灣公司貨
- 是否含基本安裝
- 壁掛安裝與線材是否另外收費
- 保固幾年，是否到府維修
- HDMI 2.1 有幾孔
- 能否接電腦跑 4K 120Hz / 144Hz
- 有沒有 PC Mode / Game Mode
- 是否支援 4:4:4 Chroma
- YouTube / Netflix 是否內建
- iPhone 能不能 AirPlay
- 手機能不能直接投 YouTube
- 是否能退換貨或現場驗機

現場實測：

- 看白底黑字，文字不能糊或有彩邊
- 移動滑鼠或切選單，延遲不能明顯
- 看賣場燈反光，不能像鏡子
- 從側邊看，顏色不要掉太多
- 開 YouTube / Netflix，系統不要卡
- 看暗場，光暈不要太誇張

## 壁掛高度

75 吋 16:9 電視大約寬 166 cm、高 93 cm。

你的使用方式是坐著看、坐沙發操作電腦，也偶爾站著看。壁掛高度應以坐著為主，站著為輔。

建議：

```text
螢幕中心高度：約 110 ~ 120 cm
螢幕底部離地：約 65 ~ 75 cm
螢幕上緣離地：約 158 ~ 168 cm
```

最簡單抓法：

> 75 吋壁掛時，底部離地約 70 cm。

不建議為了站著看，把中心拉到 140 cm 以上；坐著操作 Windows、Codex、VS Code 會變成長時間仰頭。

## Windows 建議設定

```text
解析度：3840 x 2160
Windows 縮放：200%
VS Code 字體：18 ~ 22 px
Terminal 字體：18 ~ 22 px
瀏覽器縮放：125% ~ 150%
Word 字體：16 ~ 20 pt
Excel 縮放：130% 左右
滑鼠速度：調快
工作列：可考慮放大或自動隱藏
```

電視端：

- 開啟 PC Mode 或 Game Mode
- 關閉動態補幀
- 關閉過度銳化
- 關閉降噪
- 亮度不要開太高
- 若有 HDMI 標籤設定，將接電腦的 HDMI 設為 PC
