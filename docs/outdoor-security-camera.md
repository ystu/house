# 門口戶外監視器決策

更新日期：2026-05-16

## 決策

```text
購買 2 台 Tapo C325WB
左右各安裝 1 台
```

## 必要條件

- Wi-Fi
- 可戶外使用
- 防曬
- 防水
- 左右兩側都有電源，可使用有線供電

## 候選型號

| 型號 | 定位 |
| --- | --- |
| `Tapo C325WB` | 首選；固定鏡頭、夜間畫質較強 |
| `Tapo C320WS` | 次選；固定鏡頭、規格較均衡 |
| `Tapo C310` | 入門選項；若想壓低預算可看 |
| `Tapo C530WS` | 特例；若其中一側還想兼顧車道或側巷，可改用旋轉款 |

## 三台固定鏡頭差異

| 型號 | 主要差異 | 適合誰 |
| --- | --- | --- |
| `Tapo C325WB` | 夜間畫質最好；ColorPro、較大感光元件、F1.0 大光圈、2K QHD、視角也較寬 | 想把門口一次裝好，尤其在意晚上看清楚人臉 |
| `Tapo C320WS` | 2K QHD、星光級全彩夜視，整體均衡 | 想要畫質不錯，但不一定要上到 C325WB |
| `Tapo C310` | 3MP、一般夜視，規格最基本 | 想省預算，白天監看需求大於夜間細節 |

## 目前首選

- `Tapo C325WB` 支援 Wi-Fi
- 適合戶外使用，具備防水防塵能力
- 兩顆固定鏡頭左右分工，比單顆超廣角更適合看清楚來客與包裹

## 安裝方式

```text
左側鏡頭  →  略微朝門中央
右側鏡頭  →  略微朝門中央
```

- 兩顆畫面要在門口核心區稍微重疊
- 優先拍清楚：來客臉部、門前停留區、包裹放置區
- 不需要再以單顆 180° 或電池式機種為主軸考量

## 最終結論

```text
買 2 台 Tapo C325WB，左右各一。
```
