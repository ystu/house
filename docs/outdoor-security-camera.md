# 門口戶外監視器評估

更新日期：2026-05-16

## 一句話結論

如果需求是：

- 自己安裝
- 裝在家門口
- 要能承受風吹日曬雨淋
- 需要 Wi-Fi
- 手機 App 可即時查看與回放

我會把品牌先收斂到 `TP-Link Tapo`，再依門口環境選型號：

```text
固定看門口：Tapo C325WB
門口範圍較大、需要旋轉追蹤：Tapo C520WS
預算較高、想一步到位：Tapo C560WS
完全不想拉電線：Tapo C425
```

如果只問「最推薦哪台」，我會先看：

```text
1. Tapo C325WB
2. Tapo C520WS
3. Tapo C560WS
```

## 先講判斷標準

門口監視器最重要的，不是規格表看起來最滿，而是以下幾件事：

| 判斷點 | 為什麼重要 |
| --- | --- |
| 戶外防護 | 門口長期風吹日曬雨淋，至少要有 `IP66` |
| 夜間畫質 | 真正需要辨識人臉、衣著、包裹時，常常發生在晚上 |
| 是否需要旋轉 | 固定門口若一顆鏡頭就能看完整，固定鏡頭通常更直接 |
| 供電方式 | 有線供電較適合長期守門；電池式勝在免配線 |
| 回放與儲存 | 至少要能用 App 回看，最好支援 microSD 本地儲存 |
| 偵測能力 | 人物偵測比單純移動偵測更實用，能減少誤報 |

### 固定鏡頭和旋轉鏡頭怎麼選

| 類型 | 適合情境 | 優點 | 缺點 |
| --- | --- | --- | --- |
| 固定鏡頭 | 門口範圍單純，一顆鏡頭就能完整覆蓋 | 畫面穩定、安裝後省心 | 看不到鏡頭外的區域 |
| 旋轉鏡頭 | 門口連著車道、側巷，或需要左右追蹤 | 覆蓋範圍大、可追蹤移動目標 | 體積較大，也比較不像「裝好就忘」 |

如果是一般住宅大門，我會先問自己一件事：

```text
鏡頭固定住之後，能不能把門、包裹區、來客臉部都拍清楚？
```

如果答案是可以，固定鏡頭通常就是更乾淨的解法。

## 主要候選

| 型號 | 定位 | 重點優勢 | 可能缺點 |
| --- | --- | --- | --- |
| `Tapo C325WB` | 固定主力型 | 低光源彩色夜視強、適合看門口細節、Wi-Fi、IP66 | 不能旋轉 |
| `Tapo C520WS` | 均衡旋轉型 | 2K、360° 覆蓋、人物 / 寵物 / 車輛偵測、星光級全彩夜視、IP66 | 若門口本來就很單純，功能可能用不到 |
| `Tapo C560WS` | 高階旋轉型 | 4K、Wi-Fi 6 雙頻、360° 追蹤、人臉辨識、IP66 | 對一般門口來說可能偏高規 |
| `Tapo C425` | 免配線型 | 電池式、100% 無線安裝、IP66、視角廣 | 若有電源可拉，長期省心度通常不如有線供電 |

## 各台怎麼看

### Tapo `C325WB`

這台最像「門口專用」的正解。

優點：

- 有線供電，透過 Wi-Fi 連線
- 主打低光源彩色夜視
- 適合固定看大門、包裹區、來客臉部
- `IP66` 防水防塵

注意：

- 不能旋轉
- 安裝前要先確認一顆固定鏡頭是否已足夠覆蓋門口

適合：

- 一般住宅門口
- 希望畫面穩、夜間辨識力好
- 不需要追蹤很大範圍

### Tapo `C520WS`

這台是最均衡的旋轉款。

優點：

- `2K QHD`
- 360° 視覺覆蓋
- 星光級全彩夜視
- 人物 / 寵物 / 車輛偵測
- 可用 microSD 或雲端儲存
- `IP66` 防水防塵

注意：

- 若只是單純看大門，旋轉功能不一定必要
- 機身會比固定鏡頭更有存在感

適合：

- 門口外還有車道、側邊通道
- 想把門前較大範圍一起納入
- 希望兼顧價格與功能

### Tapo `C560WS`

這台是 Tapo 戶外線裡比較接近一步到位的選擇。

優點：

- `4K 8MP`
- Wi-Fi 6 雙頻
- 360° 追蹤
- 本機人臉辨識
- 星光級全彩夜視
- `IP66` 防水防塵

注意：

- 對一般住宅門口來說，規格可能超過實際需求
- 若你不會常用人臉辨識或 4K 細節，效益未必成比例

適合：

- 預算較寬
- 希望一次買高階
- 門口範圍大，且很在意畫質與辨識能力

### Tapo `C425`

這台的核心價值，是完全不用拉電線。

優點：

- 100% 無線安裝
- 電池式
- `150°` 廣角
- `IP66` 防水防塵
- 也支援 App 與 microSD 儲存

注意：

- 若門口本來就能拉電，我不會優先選它
- 電池式的價值在安裝自由，不是在長期最省事

適合：

- 租屋
- 牆面不好拉線
- 只想快速裝起來，不想動到既有配線

## 我會怎麼選

### 一般住宅大門

```text
先選 C325WB
```

原因是門口通常是固定場景，真正重要的是：

- 人臉能不能看清楚
- 夜間畫面夠不夠好
- 長期會不會穩

這三件事比「鏡頭會不會轉」更重要。

### 門口同時要看車道或側邊

```text
改選 C520WS
```

如果固定鏡頭會顧此失彼，旋轉式就開始有價值。  
`C520WS` 也是我認為功能與價格最平衡的旋轉款。

### 想買高階、之後不想再升級

```text
選 C560WS
```

它比較像「把未來幾年的餘裕一起買進來」。  
但若只是單純守家門，這筆升級不是必要，而是偏好。

### 完全不想拉電線

```text
才選 C425
```

電池式不是不好，而是它解決的是「不能拉線」這個問題。  
如果你能拉電，固定供電通常更適合長期守門。

## 安裝前先確認

### 1. 門口 Wi-Fi 是否夠穩

監視器能不能好用，很多時候先輸在訊號。  
安裝前先在預定位置用手機測試：

- Wi-Fi 是否穩定
- 會不會常掉訊號
- 2.4GHz 是否比 5GHz 更可靠

### 2. 鏡頭高度

不要裝得太高。  
太高會變成只看到頭頂，不利辨識來客臉部。

### 3. 是否有逆光

若鏡頭正對大門外強光，白天可能會讓臉部偏暗。  
安裝前最好先用手機模擬一下畫面。

### 4. 儲存方式

如果你希望：

- 手機能回放
- 不想每月付雲端費

那就記得另外買 microSD 卡。  
Tapo 多數戶外機種都支援本地儲存。

## 最後建議

如果你現在只想先買一台，不想把事情搞複雜：

```text
首選：Tapo C325WB
若需要旋轉：Tapo C520WS
```

這兩台把大多數家庭門口的需求都照顧到了。  
`C560WS` 是升級款，`C425` 是免配線特例；兩者都不是第一直覺，而是情境成立後才往上選。
